# NAME

XString - Isolated String helpers from B

# VERSION

version 0.010

- cstring(STR)

    Returns a double-quote-surrounded escaped version of STR which can
    be used as a string in C source code.

- perlstring(STR)

    Returns a double-quote-surrounded escaped version of STR which can
    be used as a string in Perl source code.
